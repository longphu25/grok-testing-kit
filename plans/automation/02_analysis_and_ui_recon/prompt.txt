## Mission

Use **browser tool** (Playwright MCP / Selenium MCP) to:

1. **Open browser** with the URL provided (viewport 1920x1080)
2. **Navigate** step by step in Test Cases
3. **Collect** all necessary locators by:
   - Read Accessibility Tree
   - Inspect DOM attributes:`id`, `data-testid`, `name`, `aria-label`, `role`, `placeholder`
   - ABSOLUTELY **no guessing** selectors — only take from the real DOM
4. **Prioritize** locators in order (refer to `.agents/rules/locator_strategy.md`):
   - Playwright: `getByRole` > `getByLabel` > `getByPlaceholder` > `getByTestId` > CSS > XPath
   - Selenium: `id` > `data-testid` > `name` > `cssSelector` > `xpath`
5. **Return results** in table form:

| Page | Element | Locator Type | Locator Value | Fallback Locator |
|------|---------|-------------|---------------|------------------|
